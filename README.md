# Cinema Management System
A lightweight web application for managing a movie theater, built with Python, Flask, and SQLite 3. This project is well-suited for learning and practicing database operations and building basic web applications.

## Technologies Used
| Component | Technology / Tool |
| :--- | :--- |
| **Backend** | Python, Flask Framework |
| **Database** | SQLite 3 |
| **Frontend** | HTML5, CSS3, Jinja2 Template Engine |
| **Tools** | VS Code, Draw.io (ERD diagram design) |

## Folder Structure
```text
Quan-ly-rap-chieu-phim-DBD/
├── templates/          # HTML interface (Jinja2 templates)
├── app.py              # Main entry file, handles backend logic & routes
├── database.py         # Connection configuration and SQLite operation functions
├── schema.sql           # SQL script that initializes the database table structure
├── cinema.db            # SQLite database file (auto-generated after running)
└── README.md            # Usage documentation
```

## Getting Started
- Download/clone the project.

### Requirements
- Make sure you have Python 3.x installed.

```
Install Flask:
pip install flask
```

## Running the Application
```
python app.py
```
- Access it by opening your browser and going to *http://127.0.0.1:5000*
